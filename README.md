# LLM Analysis Quiz Project

## Project Overview
This project implements an API endpoint that can solve data science quizzes involving data sourcing, preparation, analysis, and visualization using Large Language Models (LLMs).

## Features
- API endpoint for quiz task processing
- Data sourcing from various formats (web, PDF, APIs)
- Data preparation and cleaning
- Data analysis and visualization
- LLM integration for complex problem solving
- Secure prompt handling and code word protection
